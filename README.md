#  PunchITNow 9.2 - Gmail Account Manager

![Version](https://img.shields.io/badge/version-9.2.0-blue.svg)
![Python](https://img.shields.io/badge/python-3.8+-green.svg)

**PunchITNow** - професійний інструмент для управління Gmail акаунтами з системою автооновлень.

---

##  Основні можливості

###  Gmail Hacks
- Account Manager з Organization Control
- Автоматичний парсинг Google Sheets
- Click-to-copy та Undo функції
- Cloudflare API інтеграція

###  Google Sheets Integration
- Налаштовувані назви колонок
- Динамічний мапінг полів
- Кешування даних

###  Card Autofill
- Автоматичне заповнення карток
- Формат: номер + дата + CVV

###  Автооновлення
- Перевірка оновлень з GitHub
- Автоматичне завантаження та встановлення
- Backup старої версії

---

##  Швидкий старт

```bash
# Клонувати репозиторій
git clone https://github.com/YOUR_USERNAME/PunchITNow.git

# Встановити залежності
pip install -r requirements.txt

# Запустити
python "New soft 3.0.py"
```
