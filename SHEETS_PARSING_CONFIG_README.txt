🎯 ЩО ДОДАНО:

1. Нова секція в Settings Tab:
   - "📊 Парсинг Google Sheets"
   - Кнопка "📊 Налаштування колонок"

2. Діалог налаштувань парсингу:
   - 11 полів для назв колонок:
     * 📝 Назва профілю
     * 📧 Email/Login
     * 🔒 Password
     * 🔐 2FA
     * 📋 Backup Codes
     * 📮 Reserve Mail
     * 💳 Credit Card
     * 📊 Конверсія
     * 🔑 API Cloudflare
     * 🆔 Cloudflare ID
     * 🔐 Cloudflare Password

3. Конфігурація зберігається в:
   config/sheets_parsing_config.json

═══════════════════════════════════════════════════════════════════════

🔧 ЯК ВИКОРИСТОВУВАТИ:

1. Відкрийте Settings → 📊 Налаштування колонок

2. Введіть назви колонок з вашої Google Sheets таблиці:
   
   Приклад 1 (українською):
   - Назва профілю: "Название"
   - Email/Login: "Почта"
   - Password: "Пароль"
   - 2FA: "2фа"
   - Backup Codes: "Бэкап"
   - Reserve Mail: "Резервка"
   - Credit Card: "Карта"
   - Конверсія: "конверсія"
   
   Приклад 2 (англійською):
   - Назва профілю: "Profile Name"
   - Email/Login: "Email"
   - Password: "Pass"
   - 2FA: "Two Factor"
   - Credit Card: "Card Number"

3. Натисніть "💾 Зберегти"

4. Тепер при імпорті з Google Sheets софт автоматично знайде ваші колонки!

⚡ АВТОМАТИЧНЕ ВИЗНАЧЕННЯ:

Софт робить наступне:
1. Читає перший рядок таблиці (заголовки)
2. Шукає назви, які ви вказали в налаштуваннях
3. Запам'ятовує, в якій колонці що знаходиться
4. При імпорті профілю підставляє дані з правильних колонок

💡 Якщо колонку не знайдено - вона просто пропускається, без помилок.
